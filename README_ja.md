## 概要

TokenPrism は、macOS のメニューバーに常駐するネイティブアプリで、Codex と Claude の API 使用状況をリアルタイムで監視します。使用量モニタリング、エージェントアクティビティ追跡、スマート残高アラート、プライバシー最優先のローカルストレージをミニマルな一目でわかるインターフェースに統合しています。

**無料・広告なし・Apple ネイティブ体験。macOS 26.5+ 対応。**

## 主な機能

### エージェント状態監視
Codex と Claude のエージェントアクティビティをリアルタイムで追跡。視覚的な状態インジケーターで一目で確認 — 青は作業中、緑は完了、赤はエラー。両方のプロバイダーでアクティビティを追跡し、レイアウトとソース順序を設定可能。

### スマート残高アラート
サイクルと週次の使用量アラートにカスタムしきい値を設定。残高が設定した制限を下回ると、TokenPrism が macOS 通知を送信。異なる使用パターンに合わせて異なるしきい値を設定 — 予期せぬ残高切れを防止。

### メニューバー統合
現在の使用率を macOS メニューバーに直接表示。残高に応じてアイコンが 6 段階で変化し、瞬時に状況を把握。オプションのパーセンテージテキストとカウントダウンタイマーで詳細も確認可能。

### プライバシーとセキュリティ
TokenPrism はローカル環境変数から API キーを読み取り — 一切のアップロードや共有はありません。すべての使用データは SwiftData でローカルに保存。アカウント不要、トラッキングなし、分析なし。
