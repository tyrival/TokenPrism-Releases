## 概述

TokenPrism 是一款常驻 macOS 菜单栏的原生应用，实时监控你的 Codex 和 Claude API 使用情况。融合用量监控、Agent 活动追踪、智能余量提醒与隐私优先的本地存储于极简的速览界面中。

**免费、无广告、Apple 原生体验。要求 macOS 26.5+。**

## 核心功能

### Agent 状态监控
实时追踪你的 Codex 和 Claude Agent 活动。视觉状态指示一目了然——蓝色表示工作中，绿色表示已完成，红色表示错误。支持跨两个提供商的可配置布局和来源排序。

### 智能余量提醒
为周期和每周用量设置自定义阈值提醒。当余量低于你设定的限制时，TokenPrism 会发送 macOS 通知。根据不同使用模式配置不同阈值 — 再也不会意外耗尽。

### 菜单栏集成
在 macOS 菜单栏中直接查看当前用量百分比。图标根据剩余余量分为 6 个等级变化，提供即时视觉提示。可选的百分比文字和倒计时让你随时掌握用量状态。

### 隐私安全
TokenPrism 从本地环境变量读取 API 密钥 — 不会上传或分享任何内容。所有用量数据使用 SwiftData 本地存储。无需账户、无追踪、无分析。
