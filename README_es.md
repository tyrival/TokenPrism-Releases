## Descripción general

TokenPrism es una aplicación nativa para macOS que vive en tu barra de menús y monitorea el uso de tus API de Codex y Claude en tiempo real. Combina monitoreo de uso, seguimiento de actividad de agentes, alertas inteligentes de saldo y almacenamiento local que prioriza la privacidad en una interfaz mínima y legible de un vistazo.

**Gratis. Sin anuncios. Experiencia nativa Apple. Requiere macOS 26.5+.**

## Funciones

### Monitoreo de estado de agentes
Sigue la actividad de tus agentes Codex y Claude en tiempo real. Indicadores visuales de estado de un vistazo — azul para activo, verde para completado, rojo para errores. Diseño y orden de fuentes configurables en ambos proveedores.

### Alertas inteligentes de saldo
Establece umbrales personalizados para alertas de uso por ciclo y semanal. Cuando tu saldo cae por debajo de tus límites definidos, TokenPrism envía una notificación de macOS. Configura diferentes umbrales para distintos patrones de uso — nunca te quedes sin saldo inesperadamente.

### Integración en la barra de menú
Ve tu porcentaje de uso actual directamente en la barra de menú de macOS. El icono cambia a través de 6 niveles según el saldo restante, dándote una señal visual instantánea. El texto de porcentaje opcional y el temporizador de cuenta regresiva te mantienen informado de un vistazo.

### Privado y seguro
TokenPrism lee las claves API de tus variables de entorno locales — nada se sube ni se comparte. Todos los datos de uso se almacenan localmente con SwiftData. Sin cuentas, sin rastreo, sin análisis.
